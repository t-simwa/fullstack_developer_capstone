# Best Cars Dealership - Full Stack Capstone Project

A full-stack dealership review web application built with Django, React, Express, and MongoDB.

## Project Overview

**Best Cars Dealership** allows users to browse car dealerships, read and post reviews, and view sentiment analysis on reviews.

## Tech Stack

- **Frontend**: React 18, React Router
- **Backend**: Django, Django REST
- **Database**: SQLite (Django), MongoDB (dealerships/reviews via Express)
- **Microservice**: Flask sentiment analyzer
- **Infrastructure**: Docker, Kubernetes, GitHub Actions CI/CD

## Features

- User registration, login, logout
- Browse all dealerships, filter by state
- View dealer details and reviews with sentiment (positive/neutral/negative)
- Authenticated users can post reviews
- Django admin for CarMake and CarModel management
